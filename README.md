# 🌟 星象馆 Stellarium Mobile

移动端专业级实时天文观测应用，支持设备方向追踪与 AR 星空体验。

## 功能特性

- 🌌 **实时星图**：基于专业天文算法，精确显示当前时间、地点的星空
- 📱 **AR 模式**：跟随设备方向移动，以设备朝向为中心显示星空
- 🪐 **太阳系天体**：太阳、月球、八大行星实时位置
- 🔭 **深空天体**：梅西耶、科德韦尔、NGC/IC 精选天体
- 📡 **专业精度**：VSOP87/ELP-2000 理论，岁差章动模型
- 💾 **离线可用**：完整 PWA 支持，首次加载后无需网络

## 技术栈

- Three.js (WebGL) - 3D 渲染
- 自研天文引擎 - 坐标转换与天体位置计算
- Vite - 构建工具
- GitHub Pages - 部署托管
